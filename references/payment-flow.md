# 支付流程 — 概念说明

用户选择套餐后，AI 按以下步骤完成交付：

1. **展示套餐** — 从指定接口获取可用套餐列表及价格
2. **创建订单** — 用户选定套餐后，AI 向后端发送订单请求
3. **生成支付二维码** — 后端返回二维码链接（`qr_code`字段），AI 生成二维码图片后用 `send_message` 发 `MEDIA:/tmp/qr.png`
4. **轮询支付状态** — 用户告知已支付后，AI 查询订单状态
5. **交付密钥** — 确认支付成功后，取出 API 密钥交付给用户

## 密钥交付格式

```
✅ 支付成功！

🔑 您的API密钥：`<密钥>`

📌 使用方式：
直接把短视频链接发给我即可自动解析
```

## 注意事项

- 密钥以文字形式展示（非图片），方便用户复制
- 提示用户直接发链接即可使用，无需其他操作
- 轮询超时后引导用户手动联系确认
- **发文件：** 用 `send_message` 工具发 `MEDIA:/tmp/xxx`，只用这一种方式，不要混用 QQ Bot API
